#  База данных "Detailing" на PostgreSQL

##  О проекте

<p> Эта база данных создана для хранения информации о машинах, клиентах, сотрудниках, материалах, услугах, участвующих в работе по уходу за автомобилями. Она позволяет вести учёт (искать автомобили по различным критериям, регистрировать клиентов и рассчитывать скидки)</p>
<p> Особенности реализации Нормализация данных: структура соответствует<strong> 3-й нормальной форме

##  Структура базы данных и связи

###  Основные сущности:

- **client** - информация о клиентах
- **car** - данные об автомобилях клиентов
- **employee** - информация о сотрудниках
- **service** - каталог услуг
- **order** - информация о заказах
- **order_service** - связь заказов и услуг
- **material** - расходные материалы
- **material_usage** - учет расхода материалов
- **service_material** - нормы расхода материалов для услуг
